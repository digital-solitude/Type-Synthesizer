About This Project

This is a project by Anisha Baid.

It is an interactive type synthesizer using p5.js and p5.sound, where typing triggers audio and visual effects.

How to Use

  • Press any key to begin.
  • Certain keys (e.g., 'q', 'w', 'e', etc.) trigger musical notes.
  • Other typed characters will be displayed on screen and create visual effects.
  • Adjust the Attack, Delay, and Distortion sliders to shape the sound.
  • Resize the window to see how the layout adapts.
